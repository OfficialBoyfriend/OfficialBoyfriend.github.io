ABB机器人小记: 实例程序(PROC)与功能函数(FUNC)

· 2020-05-14 · # 学渣笔记 # abb

PROC 实例程序

PROC main()
    TPWrite "Hi, ABB!";
 ENDPROC

将在示教器上显示：Hi, ABB!。

FUNC 功能函数

PROC main()
    ! 功能函数调用
    TPWrite hello();
ENDPROC

! 功能函数声明
FUNC string hello()
    return "Hello ABB!";
ENDFUNC

将在示教器上显示：Hello ABB!。

两者区别

  • PROC没有返回值，FUNC有返回值。
  • PROC调用时直接输入名称即可，FUCNC调用时需要在名称后加括号(())。

两者共同点

  • 支持可选参数(switch)
  • 支持值传递与引用传递(inout\var)
